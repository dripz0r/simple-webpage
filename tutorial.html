Step 1: Download & Install VS Code

VS Code is like your code BFF—free, open-source, and works on Mac, Windows, and Linux.

  • Mac / Windows / Linux: Get VS Code

Once installed, open your project folder:

cd /path/to/simple-website
code .

This fires up VS Code with your files ready to edit—no fuss.

Step 2: Install & Use Live Server

  1. In VS Code, click the Extensions icon (or press Ctrl+Shift+X / Cmd+Shift+X).
  2. Search for Live Server and click Install.
  3. Open tutorial.html, then click Go Live in the bottom-right.

Your browser opens http://127.0.0.1:5500/tutorial.html and auto-refreshes whenever you save. 🎉

Step 3: Edit & Preview

Now you can tweak tutorial.html and style.css right in VS Code. Hit Ctrl+S / Cmd+S, and watch the magic—no extra puppetry needed.

Step 4: Try an Online Editor (No Install)

If you don’t want to install anything, paste your code into one of these live playgrounds:

  • CodePen — Create a new Pen, paste HTML/CSS, see live updates.
  • StackBlitz — Edit on the right, preview on the left.
  • Replit — Code and preview side-by-side.

Step 5: Style Tips & Next Moves

  • Colors: Stick to one background, one text, one accent for a clean vibe.
  • Fonts: Sans-serif fonts keep it simple and modern.
  • Spacing: Add margin and padding so elements breathe.
  • Want more pages? Duplicate tutorial.html as page2.html and update links.

There you go—your own tutorial page, ready for any curious mind to follow. 🚀
